void SetCanvasSortingOrder (int sortingOrder)
 

Detailed Description

Dropdowns menus in Unity are automatically set to sorting order 30000, which does not play nicely with world space UIs. Meant to be used in conjunction with an EventTrigger on a given Dropdown, this component can be used to set a different sorting order on this and any child canvas.
