Generates a new HandGrabPoseData that mirrors the provided one. Left hand becomes right hand and vice-versa. The mirror axis is defined by the surface of the snap point, if any, if none a best-guess is provided but note that it can then moved manually in the editor.

Parameters
originalPointThe point to mirror
mirrorPointThe target HandGrabPose to set as mirrored of the originalPoint
