static Texture2D Oculus.VR.Editor.OVRSystemSplashScreenEditor.ProcessTexture ( Texture2D  texture)
static

Utility method that tries to make a blit copy of the input texture, and outputs its content in the suitable file encoding & format to the default path.

If it fails (e.g. No GPU), it will return the original input texture instead.

Parameters
textureInput texture to make a copy of
Returns
The copied texture, or the input texture if the operation fails
