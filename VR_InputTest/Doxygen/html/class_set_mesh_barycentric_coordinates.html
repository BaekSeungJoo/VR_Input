Detailed Description

This class will continuously check the mesh of a MeshFilter component and if it has changed to a new mesh, it will create barycentric coordinates and set this to the per vertex color information of the mesh.

We only need barycentric coordinates for our wireframe shader.

Using this will increase the number of vertices by 3. Avoid using this class for anything other than debugging.

The alternative is to use geometry shaders, however, they aren't well supported for non Multi-Pass rendering modes.
