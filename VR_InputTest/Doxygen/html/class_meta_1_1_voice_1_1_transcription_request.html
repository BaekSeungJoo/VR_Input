Detailed Description

Abstract class for all transcription requests.

Template Parameters
TUnityEventThe type of event callback performed by TEvents for all event callbacks
TOptionsThe type containing all specific options to be passed to the end service.
TEventsThe type containing all events of TSession to be called throughout the lifecycle of the request.
TResultsThe type containing all data that can be returned from the end service.
Type Constraints
TUnityEvent :UnityEventBase 
TOptions :ITranscriptionRequestOptions 
TEvents :TranscriptionRequestEvents<TUnityEvent>
