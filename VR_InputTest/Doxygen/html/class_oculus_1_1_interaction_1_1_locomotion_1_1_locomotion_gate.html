Detailed Description

This Gate reads the Hand orientation towards the shoulder and decides if it should be in Teleport mode or Turning mode. It enables/disables said modes based on some Input ActiveStates (EnableShape and DisableShape). It outputs it result into two ActiveStates (for Teleport and Turn)
