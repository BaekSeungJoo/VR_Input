Oculus.Interaction.HandGrab.HandGrabPose Class Reference

The HandGrabPose defines the local point in an object to which the grip point of the hand should align. It can also contain information about the final pose of the hand for perfect alignment as well as a surface that indicates the valid positions for the point. More...

Detailed Description

The HandGrabPose defines the local point in an object to which the grip point of the hand should align. It can also contain information about the final pose of the hand for perfect alignment as well as a surface that indicates the valid positions for the point.
