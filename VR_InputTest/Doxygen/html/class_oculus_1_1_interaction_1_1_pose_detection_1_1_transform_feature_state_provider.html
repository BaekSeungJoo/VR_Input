Detailed Description

Interprets transform feature values from a TransformFeatureValueProvider and uses the given TransformFeatureStateThresholds to quantize these values into states. To avoid rapid fluctuations at the edges of two states, this classes uses the calculated feature states from the previous frame and the given state thresholds to apply a buffer between state transition edges.
