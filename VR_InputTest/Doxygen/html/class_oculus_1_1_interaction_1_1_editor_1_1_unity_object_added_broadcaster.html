static void Oculus.Interaction.Editor.UnityObjectAddedBroadcaster.HandleHierarchyChanged ( HashSet< int >  knownIds)
staticprivate

Fires signals for GameObjects and Components added through the addition of a prefab, checking whether the selected GameObject at the moment of a hierarchy change is unfamiliar (i.e., has an instance ID which is not already in known IDs) and signaling appropriately. Note that this will NOT signal GameObjects or Components added to the scene through prefab updating, which are added without modifying the Editor's selection variable, upon which this handler relies.

Parameters
knownIdsCache of known GameObject instance IDs
